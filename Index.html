    2.How to Create Repositories in GitHub.

      A.Create GitHub Account.

      B.Go to "HomePage" of GitHub.

      C.Click on "Profile",pick to get the suggestions.

      D.Go to "Your Repositories".

      E.Click on "New".

      F.Fill "Repository Name".

      G.Write "Description".

      H.Click on "Public"(Give access to EveryOne).

      I.Initialize this repository with:Click on "Add a README file".

      J.Click on "Create Repository".
